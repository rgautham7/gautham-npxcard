# Gautham - NPX Card

> A fancy, interactive terminal card for **R Gautham** — right inside your CLI.

Turn your terminal into a mini-portfolio experience and make a memorable first impression with a single command.

---

## Usage
```bash
npx gautham
```

## 🎯 What You Get

An immersive CLI experience featuring:

* **🎨 Gradient ASCII Name Art**: Beautifully rendered name with smooth, vibrant color transitions.
* **📜 Scrolling Marquee Intro**: A dynamic introduction that feels alive and engaging.
* **🎮 Interactive Navigation**: Use your arrow keys to explore different sections seamlessly.
* **🔗 Quick Actions**: Instantly launch external links:
    * **Portfolio** 🌐
    * **Resume** 📄
    * **Email** 📬
* **⚡ Zero Install**: Runs directly via `npx`—no clutter, no global installs.

---

## 🛠️ Tech Stack

Built with the best-in-class Node.js CLI tooling:

* **chalk** — Terminal string styling
* **figlet** — Large ASCII text banners
* **gradient-string** — Smooth color gradients for text
* **inquirer** — Interactive user prompts
* **boxen** — Clean UI boxes and borders
* **ora** — Elegant terminal spinners
* **open** — Launching URLs in the browser

## 🧩 How It Works

1.  **Fetch**: `npx` pulls the package from the registry.
2.  **Render**: The script initializes styled ASCII layouts and UI components.
3.  **Interact**: An event-driven menu waits for your input.
4.  **Execute**: Selected actions trigger shell commands to open links or display data.
